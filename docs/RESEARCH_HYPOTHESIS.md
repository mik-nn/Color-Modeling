# RESEARCH_HYPOTHESIS.md

## Основная гипотеза (Device-Substrate Separation)

При неизменных условиях печати (device conditions) **относительные пропорции поведения красок** между каналами сохраняют линейную или афинную переносимость при смене субстрата.

### Формальная постановка

Пусть есть две функции предсказания цвета:
- \( f_d(\mathbf{c}) \) — поведение устройства (device)
- \( g_s(\cdot) \) — влияние субстрата

Тогда измеренный цвет можно представить как:
\[ \mathbf{color} = g_s(f_d(\mathbf{c})) \]

Гипотеза утверждает, что \( g_s \) можно аппроксимировать **афинным преобразованием** в подходящем цветовом пространстве (лучше всего — спектральном или XYZ).

### Подгипотезы

1. Residuals после модели Ink Spreading YNSN линейно коррелируют между субстратами.
2. Относительные ink spreading coefficients (slope и bias между каналами) стабильны.
3. Большая часть различий объясняется Media White Point + глобальным scaling-фактором.
4. CIECAM02-подобная адаптация улучшает результаты по сравнению с простым Media Relative scaling.

### Метрики подтверждения
- Pearson/Spearman correlation > 0.93
- R² линейной регрессии residuals > 0.90
- Средний ΔE₀₀ после коррекции < 1.5 при использовании ≤ 12 патчей
